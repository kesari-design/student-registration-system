# Student Registration System

A foundation project built to learn Spring Boot fundamentals, including layered architecture, 
CRUD operations, Bean Validation, Exception Handling, REST APIs, Thymeleaf, and MySQL integration.

- Add a new student
- View student details
- Update student information
- Delete student records
- Input validation using Bean Validation
- Custom exception handling
- Global exception handling
- REST API support
- Thymeleaf-based frontend
- MySQL database integration

## Technologies Used

- Java 17
- Spring Boot 3
- Spring Data JPA
- Thymeleaf
- MySQL
- Maven
- Hibernate
- Bean Validation
- Git & GitHub

## REST API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | /fetch | Get all students |
| GET | /fetch/{id} | Get student by ID |
| PUT | /update/{id} | Update student |
| DELETE | /delete/{id} | Delete student |
